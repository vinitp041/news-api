Prerequisites

    Download and install Node.js: Node.js Official Website
    Setup Your React Project

If you don't have a React project set up, create one using create-react-app. 

Open your terminal and run:
      "npx create-react-app news-website"

Change into the project directory:
       "cd news-website"

Install Dependencies
In the terminal, 
run:
       "npm install"

Run Your React App
Install date-fns:
        "npm install date-fns"


After installing the dependencies, 
run:
         "npm start"
         
This command will start your React development server, and you should be able to view your news website at http://localhost:3000 in your web browser.
